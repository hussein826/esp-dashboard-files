# 🚦 ESP8266 Traffic Light Dashboard

A smart IoT dashboard used to monitor and control a traffic light system connected to an ESP8266.

This project was developed as part of a smart emergency traffic management system.

---

## 📌 Features

- Real-time traffic light status
- Emergency vehicle detection using RFID
- Manual override from web dashboard
- Communication with ESP8266
- Web interface for monitoring traffic

---

## 🛠 Technologies Used

- ESP8266
- Arduino
- HTML
- CSS
- JavaScript
